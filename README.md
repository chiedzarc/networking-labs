# Networking Labs (Cisco Packet Tracer)

This repository contains a collection of networking simulations created using Cisco Packet Tracer. Each lab focuses on a specific networking concept, following a structured learning path from basic to intermediate networking topics.

---

## Overview

These labs are part of a guided learning journey covering fundamental networking concepts such as:

- Device connectivity
- IP addressing
- LAN configuration
- Basic routing concepts
- Network troubleshooting

Each lab builds on the previous one to reinforce practical networking and problem-solving skills.

---

## Tools Used

- Cisco Packet Tracer

---

## Repository Contents

- `.pkt` files (network simulation projects)
- Screenshots of each lab setup
- This README file for documentation

---

## Learning Goals

By completing these labs, I aim to:

- Understand how networks are built and structured
- Practice configuring devices in a simulated environment
- Improve troubleshooting and logical thinking skills in networking
- Build a strong foundation for advanced networking concepts
